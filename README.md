PHP7.1-FPM/NGIX/MySQL/PhpMyAdmin Docker Container
=================================================

### Intro

Includes:

- composer  
- memcached  
- redis  
- xdebug  

Commands:

`docker-compose up`  
`docker-compose down`  
`docker-compose build`  
`docker-compose stop`  

AIO

`docker-compose stop && docker-compose down && docker-compose build && docker-compose up`
